## 🍎 iOS版特別注意事項

### 📋 必要な開発環境
- **Xcode 15.0以上**
- **iOS 13.0以上のデバイス**
- **Apple Developer Account** (実機テスト用)

### 🔧 セットアップ手順
1. ZIPファイルを解凍
2. `ios/Runner.xcworkspace`をXcodeで開く
3. Bundle Identifierを変更
4. 開発チーム設定
5. 実機またはシミュレータでビルド

詳細手順: `release/ios/INSTALL_iOS.md`を参照

### 🛡️ プライバシー機能
- **Photos権限**: エクスポートファイル保存用
- **Notifications**: 予算アラート用
- **Files Access**: CSV import/export用
